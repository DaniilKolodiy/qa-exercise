1 - Download or clone the repo from: https://github.com/DaniilKolodiy/qa-exercise

2 - Open the TestProject.sln file in MS Visual Studio.

3 - Resgtore Nuget Packages

4 - Build the solution. 

 Running tests
1 - Open the Test Explorer window from Test->Windows->Test Explorer menu. The 'Test Explorer' window will list the tests in the solution so that they can be run. The 6  tests displayed.

2 - The tests can be run either by using the 'Rull All' or by right clicking a test and selecting the option 'Run Selected Tests'
